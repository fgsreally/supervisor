# Mockup

为每个 capability 创建 specs/<capability>/mockup.html，不得省略此阶段。

- UI capability：展示核心页面、身份操作线、条件分支以及最小的 modal/popover 交互。
- API capability：用 HTML 清晰展示方法、路径、请求、响应、字段、错误码和示例。
- CLI capability：用 HTML 展示 usage、参数、示例和输出。

使用项目技术栈约定的组件风格；小样只用于确认交互和表达，不实现真实业务。遇到审美、交互顺序或信息层级选择时主动使用 ask。所有 spec capability 都必须存在对应 mockup.html。
